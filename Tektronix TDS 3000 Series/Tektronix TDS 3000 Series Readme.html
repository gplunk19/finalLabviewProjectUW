Tektronix TDS 3000 Series Instrument Driver Readme

To rate this driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Tektronix
Supported Language(s): LabVIEW 8.0
Supported Model(s): Tektronix TDS 3012, 3012B, 3014, 3014B, 3024B, 3032, 3032B, 3034, 3034B, 3044B, 3052, 3052B, 3054, 3054B, 3064B, 3012C, 3014C, 3032C, 3034C, 3052C, 3054C
Model(s) Tested: TDS 3054, TDS 3054B, TDS 3064B, TDS 3054C
Interface(s): GPIB, Serial, Ethernet
Firmware Revision Tested: v3.26, v3.35 (TDS 3064B only), v3.59

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 2.3
Original Release Date: 05/16/2003
Current Revision Date: 04/30/2008

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 3.0 or later
LabVIEW 8.0 or later


For more details, refer to the LabVIEW Help under Help » VI, Function, & How-To Help.

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.

4. Revision History

The latest version of this and other LabVIEW and LabWindows/CVI instrument drivers can be downloaded at the Instrument Driver Network.

REV 2.00, 7/13/05

Modified by: Rob Hammond, Austin,TX.
Original release for LabVIEW 8.0 created from v1.7 of LabVIEW 7.1 tktds3k driver.

REV 2.1, 10/23/2006

Modified by Jim Piotrowski, Austin TX

-Added Transfer Current Waveform Example.vi -Updated Fetch Waveform.vi and Fetch Min-Max Waveform.vi so that DAT:WID? returns the data size instead of using the VISA User Data property.

-Updated Initiate.vi so that the VISA User Data property isn't used

-Removed ACQ:STATE 0 and ACQ:STOPA SEQ so that tranferring the current waveform doesn't halt continuous acquisition

-Bumped the version in Revision Query.vi

REV 2.2, 6/13/2007

Modified by Vinny LaPenna, Austin TX

-Added Wait for Acquisition Complete.vi and Wait for Acquisition Complete Example.vi

Modified by Berk Ozdogan, Austin TX

- C revision instruments are now supported.

- Configure TV Trigger Source VI: Added HDTV support

- Updated control documentation in order to include important information regarding HDTV triggering
